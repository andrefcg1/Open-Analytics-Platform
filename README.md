# Open-Analytics-Platform
Construir uma plataforma analítica moderna de ponta a ponta, usando ferramentas gratuitas ou de baixo custo, com ingestão de dados via Python, armazenamento em DuckDB local e Snowflake opcional, transformação com dbt, orquestração com Airflow, documentação automática, testes de qualidade, dashboard em React e deploy gratuito.
